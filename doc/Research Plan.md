**：对于位于非编码调控区域的VUS，能否用轻度CSG解析其结构，从而为VUS的重新分类提供证据？**



遗传咨询问题——最直接的行动困境

这是VUS带来的最具体、最紧迫的临床决策困境。一旦涉及家系和下一代，VUS就从一个“学术不确定”变成了“行动不确定”。



或是



预后问题——患者最关心的

患者真正想知道的是：“这个变异对我意味着什么？我会不会得病？什么时候得？多严重？”



自己认为两者是重要的。



因此现在我们可以将我们的问题归类到 VUS分类+决策问题。而句法解析结果作为证据。

## 句法解析能提供什么性质的证据

关键区分：

```
现有方法提供的：
  统计证据     "在已知致病变异中，这种模式出现了 X 次"

句法解析提供的：
  结构性证据   "这个变异破坏/保留了文法中的哪条规则"
               "破坏的是终结符替换（同义）
                还是非终结符节点（非同义）
                还是整棵子树（frameshift/剪接破坏）"
```

这是**证据性质不同**，不是精度更高。

------

## 这对两个决策困境的意义

### 遗传咨询

```
VUS 在家系中共分离的解释困难：

现在：
  "这个 VUS 在患病亲属中也存在，
   但我们不知道这是巧合还是因果"

句法证据加入后：
  "这个变异破坏了 CodingRegion 的剪接规则，
   产生移码，parse tree 在 Exon 7 处崩溃，
   共分离 + 结构破坏 → 共同支持 Likely Pathogenic"

句法结果作为 ACMG PS3（功能证据）的形式化替代
```

### 预后

```
现在：
  "BRCA1 VUS，致病概率 0.6，建议密切随访"

句法证据加入后：
  "变异保留了蛋白主体结构（parse tree 骨架不变），
   仅影响 TFBS 结合位点，
   → 功能影响可能是调控层面而非编码层面
   → 外显率可能低于完全功能丧失的已知致病变异
   → 更精准的预后区间"
```

------

## 因此我们的定位

```
我们不是要替代 ACMG 分类框架

我们是：

  为 VUS 提供一类新性质的证据
  ── 结构性证据（句法解析结果）
  ── 与现有统计证据互补
  ── 可解释、可向患者陈述
  ── 在遗传咨询和预后两个决策点上
     将"行动不确定"降低为"有依据的推荐"
```
